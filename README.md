# Installing
## Windows
1. update winget (via MS store)
2. install Git for Windows via winget
3. install pwsh via winget
4. set execution policy
5. install [scoop](https://scoop.sh/) (see scoop website)
6. run `bin/install.ps1`

## Linux
1. install fish
2. run install.fish
